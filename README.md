# Diyo Interview Prep

Private Vue + Vite interview preparation library built from the supplied interview transcript.

## Local development

```bash
npm install
npm run build:data
npm run dev
```

## Production build

```bash
npm run build
```

## Content pipeline

- source transcript: `content/interview-transcript.txt`
- generator: `scripts/build-interview-data.mjs`
- app data: `src/data/interviewContent.js`
